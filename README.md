# CalSim3 Central Valley Water Map

An interactive HTML map visualizing California's Central Valley water allocation system as simulated by CalSim3, October 1921 – September 2021 monthly, across **43 COEQWAL scenarios** (20 management themes × historical / moderate / severe climate-change hydrology). Built for a public audience.

**View the map:** serve the repo folder over http (e.g. `python3 -m http.server`) and open `CalSim3_water_map.html`, or publish via GitHub Pages. Internet required for basemap tiles and Leaflet CDN. Opening the file directly (double-click) also works and shows the default scenario (s0020 baseline, embedded); switching scenarios requires the `scenarios/` folder to be served alongside it.

## Features

Flow-scaled river/canal/bypass lines, reservoir circles with capacity rings, animated flow-direction particles (red on reversal), Banks/Jones export triangles, demand-unit delivery choropleth, water-year-type timeline with month scrubber, four guided story tours, click-any-feature 100-year charts, and a scenario picker (theme × climate hydrology) in both the big-picture and detailed views. Flow widths, reservoir scaling, and particle speeds are normalized to the baseline scenario, so visual size is comparable across scenarios.

## Data layout

- `CalSim3_water_map.html` (~6.5 MB) — self-contained map with geometry and the baseline scenario (s0020) embedded.
- `scenarios/sNNNN.json` (~5 MB each, ~1–2 MB gzipped in transit) — per-scenario time series, fetched on demand when a scenario is selected. Generated by `build/build_scenario.py`; feature set and join keys are frozen from the base payload.
- `build/scenario_meta.json` — scenario catalog (theme labels, families, hydrology variants → run IDs) from the COEQWAL scenario listing; embedded into the HTML by `make_html.py`.

## Rebuilding from source data

The large source files are not committed (they exceed GitHub's file size limit):

- `s0020_coeqwal_calsim_output.csv` (~279 MB) — wide-format DSS export from COEQWAL scenario s0020
- `20221103v2_cs3_cvhydroregion.gpkg` (~268 MB) — CalSim3 arcs, nodes, and reservoirs (EPSG:3310)
- additional `sNNNN_coeqwal_calsim_output.csv` exports for the other scenarios

Place the first two in the repo root, then:

```bash
pip install pyproj pyshp pandas
python3 build/rebuild_data.py       # geometry + baseline series → build/payload.json
python3 build/simple_mode_data.py   # baseline annual water balance → build/simple_payload.json
python3 build/make_html.py          # injects payloads + scenario catalog → CalSim3_water_map.html
python3 build/build_scenario.py path/to/sNNNN_coeqwal_calsim_output.csv   # → scenarios/sNNNN.json
```

Scenario extraction joins by variable *name*, not column position — the scenario CSV exports differ in column count (17k–24k). Verified to reproduce the embedded baseline byte-identically from the s0020 CSV.
